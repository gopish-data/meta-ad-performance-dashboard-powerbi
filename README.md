# Meta Ad Performance Dashboard (Power BI)

## About This Project
This project is a Power BI dashboard created to analyze Meta (Facebook & Instagram) advertisement performance.

The main purpose of this dashboard is to track campaign performance, engagement metrics, and conversion-related KPIs in a clear and interactive way.

## What I Did in This Project
- Imported ad performance dataset into Power BI
- Cleaned and transformed data using Power Query
- Created calculated measures using DAX
- Designed KPI cards for important metrics
- Built interactive visuals and slicers
- Created separate views for Facebook and Instagram

## KPIs Tracked
- Impressions
- Clicks
- Shares
- Comments
- Purchases
- Engagements
- CTR (Click Through Rate)
- Engagement Rate
- Conversion Rate
- Purchase Rate
- Total Budget
- Average Budget per Campaign

## Dashboard Features
- Platform selection (Facebook / Instagram)
- Weekly & Hourly purchase trend
- Purchases by Gender
- Purchases by Age Group
- Purchases by Country (Map view)
- Ad type performance analysis (Carousel, Stories, Image, Video)
- Dynamic filters for:
  - Campaign Name
  - Target Interest
  - Platform
  - Month selection

## Insights Observed
- Certain ad types generate higher CTR and engagement
- Conversion rate varies across campaigns
- Purchases are influenced by platform and audience targeting
- Budget allocation impacts engagement and purchase rate
